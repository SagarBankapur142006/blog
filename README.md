# Inkly

Inkly is a full-stack writing and social publishing web app.

## Live Demo
Frontend: https://blog-9y3anaj9b-sagarbankapur142006s-projects.vercel.app

## Tech Stack
- React
- Node.js
- Express
- MongoDB Atlas
- Vercel
- Render

## Features
- User signup and login
- Create posts
- View feed
- Like and unlike posts
- Profile page
- Discover page
- Search users

## Project Structure
- `inkly` -> frontend
- `inkly-backend` -> backend

## Notes
The deployed backend is on Render free tier, so the first request may be slow after inactivity.
